## Short Description (132 chars max)

Save DocSend presentations as PDF. Automatically captures every page and compiles them into a downloadable PDF file.

## Detailed Description

Save DocSend presentations as PDF files with one click.

DocSend presentations don't allow downloading by default. SFVibe captures each page visually and compiles them into a single PDF that you can save, share, and read offline.

HOW IT WORKS:
1. Open any DocSend presentation
2. Click the extension icon
3. Click "Start Capture"
4. The extension automatically navigates through every page
5. Your PDF downloads when complete

FEATURES:
- One-click capture — no manual screenshotting
- Automatic page navigation
- Resume interrupted captures
- Real-time progress bar
- Works with any DocSend presentation

PRIVACY:
- All processing happens locally in your browser
- No data is collected, transmitted, or shared
- No external server communication
- No analytics or tracking

This extension is open source: https://github.com/YOUR_USERNAME/sfvibe-docsend-to-pdf
